# ESP32 Home Automation System

A **web-based home automation system** built using **ESP32 firmware** and **HTML, CSS, JavaScript**.  
This project allows users to **control home appliances remotely over a public network** using a **secure login system (email & password)**.

The project is under **active development**, and updates are pushed regularly.
